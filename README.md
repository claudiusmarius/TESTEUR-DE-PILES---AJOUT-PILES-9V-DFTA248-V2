# DFTA248V2

Nouvelle version du projet **DFTA248 – Testeur de piles AA / AAA / 9V**.

Cette version V2 apporte :

- ✅ Détection 9V plus robuste (basée sur mesure ADC brute)
- ✅ Seuils spécifiques dédiés aux piles 9V
- ✅ Structure logicielle conservée

---

## 🔗 Projet original

DFTA248 (version initiale) :  
https://github.com/claudiusmarius/TESTEUR-DE-PILES-REALISATION-PCB-DFTA248

---

## 🎯 Objectif de la V2

Améliorer la fiabilité du diagnostic 9V tout en conservant :

- L’architecture matérielle
- La logique des 3 barrettes LED
- Le fonctionnement "one shot"
- La calibration personnalisée K_DIV2

---

## ⚙️ Compatibilité

- PCB DFTA248
- ATtiny
- NeoPixel 12 LEDs
- Résistance de charge 2.2Ω
